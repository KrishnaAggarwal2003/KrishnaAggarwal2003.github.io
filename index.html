About Me

I am an Integrated BS-MS graduate student from IISER Mohali (Biology Major, Data Science Minor, CPI: 8.6/10), with my MS thesis completed at the Computational Neuroscience Laboratory, IIT Madras, under Prof. V. Srinivasa Chakravarthy. My work sits at the intersection of dynamical systems, deep learning, and neural data analysis — studying how computation arises in the brain through the structure of neural dynamics.

My recent projects focused on EEG-based neural decoding using oscillatory neural networks, where temporal and oscillatory dynamics are treated as computational primitives rather than features to average away. I have a growing interest in latent population dynamics, state-space modelling, and how the time-evolution of neural activity reflects underlying cognitive computation.

Research Experience

EEG-based Cursor Navigation with Deep Oscillatory Neural Networks

MS Thesis  ·  Computational Neuroscience Lab, IIT Madras  ·  PI: Prof. V. Srinivasa Chakravarthy  ·  Aug 2025 – Apr 2026
  • Built Dense Oscillatory Neural Network (DONN) and topomap-based Oscillatory CNN architectures using Hopf-bifurcation oscillator layers, treating temporal dynamics as a learned computational primitive rather than a feature to average away.
  • Benchmarked these against EEGNet, CNN-LSTM, CNN with PSD features (XGBoost), and Riemannian (covariance and tangent space) pipelines on 128-channel EEG from 20 subjects, using Leave-One-Block-Out cross-validation for subject-level generalisation.
  • Designed and ran full data acquisition: custom MATLAB protocol with NetStation SDK integration for millisecond-precision event marker synchronisation, plus a preprocessing pipeline with band-pass filtering, ICA-based artifact removal, and epoch-level normalisation.
  • Studying whether dynamical priors improve robustness and generalisation under high trial-to-trial variability — a question relevant to how dynamical structure supports robust neural computation.

Code and data are in preparation — finalised files will be released soon.

Reconstructing STN-LFP Signals Using Neural Networks

Computational Neuroscience Lab, IIT Madras  ·  PI: Prof. V. Srinivasa Chakravarthy  ·  May 2024 – Jul 2024
  • Built coupled Rössler oscillator networks with Hermitian complex-weight coupling as a generative model for subthalamic nucleus LFPs from Parkinson's patients, with synchronisation, phase coupling, and beta-band dynamics as the core computational primitives.
  • Compared feedforward, LSTM, and Flip-Flop RNN architectures for mapping oscillatory dynamics onto experimental signals, evaluating reconstruction in both time and frequency domains.
  • Showed that recurrent and dynamical-systems-inspired architectures preserve the 13–30 Hz beta-band synchronisation peak, where purely feedforward models lose this temporal structure entirely.

Computational Analysis of Memory Load in Sentence Comprehension

Cognitive Science Department, IIT Kanpur  ·  Nov 2024 – May 2025
  • Built computational pipelines on multilingual Universal Dependencies data to extract structural features — dependency length, intervening elements, sentence complexity — and quantify their effect on memory load using linear mixed-effects models.
  • Analysed how structural constraints shape information processing across 23 languages, linking formal linguistic structure to processing difficulty. Resulted in a first-author preprint and two conference acceptances.

Computational Behavioural Analysis: Systems Neuroscience

HM Lab, IISER Mohali  ·  Sept 2024 – Dec 2024
  • Developed Python/OpenCV pipelines to extract behavioural variables from rodent video recordings in a systems neuroscience experiment, including motion tracking and pose-based feature extraction.

Synchronisation in Coupled Oscillators (Kuramoto Model)

Spatio-Temporal Chaos in Coupled Map Lattices

Studied pattern formation, travelling waves, and chaotic dynamics in nonlinear discrete systems with local coupling.

Motion Detection and Behavioural Analysis Pipeline

Developed a computer vision pipeline using OpenCV to extract and analyse motion patterns from video data for systems neuroscience experiments.

Image Classification with CNN Models

Implemented and benchmarked a feedforward ANN and two CNN architectures (decreasing and increasing filter widths) on the CIFAR-10 dataset, with parallel implementations in both TensorFlow/Keras and PyTorch for direct framework comparison.

Question Identification using Text Vectorization and ML Classifiers

Identified questions in text by combining four vectorisation techniques (TF-IDF, word embeddings, LDA, LSA) with four classical ML classifiers (Logistic Regression, Bernoulli Naive Bayes, Random Forest, SVM), comparing all 16 pipelines using ROC curves.

Regression Algorithms from Scratch

Implemented linear and logistic regression from scratch in NumPy with manual gradient descent, L1/L2 regularisation, and early stopping, with full evaluation via MSE/MAE/R² (linear) and confusion matrix/ROC-AUC (logistic).

MLP for Digit Recognition

Built a multi-layer perceptron for MNIST digit classification with both ReLU and Sigmoid activations using PyTorch, plus a separate from-scratch NumPy implementation including momentum-based backpropagation.

Cellular Automata Simulations

Simulated three families of cellular automata: Wolfram's 256-rule elementary 1-D CA, Conway's Game of Life, and a stochastic 2-D SIRS epidemic model with susceptible-to-infected transitions driven by neighbour density.

Nonlinear Dynamical Systems

Studied the Rössler system (with parameter-sweep bifurcation diagrams across all three parameters) and the Lorenz system, illustrating limit-cycle and point attractors and chaotic regimes in low-dimensional nonlinear dynamics.
